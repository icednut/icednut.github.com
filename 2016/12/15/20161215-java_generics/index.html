Wild Card

  • 오브젝트의 기능이 담긴 타입의 객체를 사용하겠다는 의미

위 두 메소드의 차이점은 뭘까?

List list = Arrays.asList(1,2,3);
printList1(list); // 이 때는 컴파일 오류가 발생함. List는 List
